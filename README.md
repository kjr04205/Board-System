<h1 align="center">Today's me "오늘의 나" 📝</h1>     

### 프로젝트를 시작하게 된 이유
> 오늘의 나의 하루를 마무리하며 습관처럼 가까이 하는 휴대폰과 컴퓨터로 기록하기 위해 만들게 되었습니다.

### 프로젝트 설명
> 1. 회원 데이터베이스 구축<br>
> 2. 회원가입, 로그인, 로그아웃 기능 개발<br>
> 3. 접속한 회원의 세션 관리 기능 개발<br>
> 4. 게시판 데이터베이스 구축<br>
> 5. 글 쓰기, 글 수정, 글 삭제, 글 보기 기능 개발<br>
> 6. 메인 페이지 Image Slider 기능 개발<br>
